⏰ PHASE 4: Attendance System (CORE)

Step 11: Attendance Model
{
  userId,
  date,
  checkIn,
  checkOut,
  workingHours
}

Step 12: Check-in API

Logic:

Get today’s date
Check if record exists ❌
If yes → reject
Else → create new attendance

Step 13: Check-out API

Logic:

Find today’s attendance
If not found → error
If already checked out → reject
Else:
Set checkOut
Calculate working hours

Step 14: Time Calculation (IMPORTANT)

In utils/timeHelper.js:

const diff = (checkout - checkin) / (1000 * 60 * 60);
